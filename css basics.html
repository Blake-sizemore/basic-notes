css basics

basic way to link a css file to an html file is to add an "link:css" to the head of the html doc.

The cascading styles are based off of parent>child relation ship. If you make body { color: red } all other tags inside of body will turn red by default.

Selectors

  • Elements
    • Any and all tags used in html
    • ie: h1-h6, a, body, section, article
  • Classes
    • Are any title (you can make them up) that starts with a .
    • Can be applied to mutiply elements
    • Be descritive: .text-center is good example
    • using more than one class on a tag is possible, instead of adding to many rules to single class
      • class="text-center text-red" : use a space between each additional class

  • ID's
    • Are any title (you can make them up) that starts with a #
    • Can be applied to single elements, exist only ONCE per page
    • Best practice for ID naming is camel case: submitButtonRed
  • Attributes
    • Least common selector that finds and is aimed directly at specific attributes of elements
    • uses [] as it call out: [for="email"]
