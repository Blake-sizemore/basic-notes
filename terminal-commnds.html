List of learned commands and functions

Information and location discovery

  • man - manuel for all a commands
  • pwd - print working directory
  • show me where this terminal is working from ( show where your working directory are )
  • ls - list
  • shows contents of directory (shows whats in the location I am at)
  • cd - change directory - change directory to select items in current directory
  • cd .. - go up one folder ( .. means go to parent folder )

Make,Move,Copy,Delete,Rename

  • mkdir - make directory
  • make a folder
  • touch - make file
  • - touch filename.txt
  • Mv - move folders to rename folders
    • mv filename1.txt filename2.txt (changes file names)
    • mv filename1.txt .. (pull up a folder)
    • mv filename1.txt . folder2 (moves file into folder2 )
  • Cp - copy a file and possible rename
  • Rm - remove - NO UNDOING
  • only for files
  • - deletes folders and contents inside of folder NO UNDOING
  • Code - will open visual studio code (C no location associated expect current PWD)
  • - code . - opens current directory in terminal ( . means current folder)
